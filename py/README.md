# APatch Auto Patch Tool Python Version

## Usage

1. Clone this repo:

```shell
git clone https://github.com/AkinaAcct/APatchTool.git
```

2. Install deps:

```shell
cd py
pip install -r requirements.txt
```

3. Run:

```shell
python AAP.py -h # Get usage
```
